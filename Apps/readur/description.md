# Readur

Readur is a self-hosted read-it-later and document archiving service. Save articles, PDFs, and web pages; read them later with full-text search and offline access. A lightweight alternative to Wallabag.

**First steps:** Open `http://<host>:8000` and register your account.

**Data:** `/DATA/PowerLabAppData/readur/` — Archived articles, PDFs, and full-text search index.
