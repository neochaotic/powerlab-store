# mStream

mStream is a personal music streaming server with a clean web player, mobile apps, and Subsonic API support. Stream your local music library to any device. Supports playlists, scrobbling, and folder-based browsing.

**First steps:** Open `http://<host>:3000` — default login is `admin` / `admin`. Change immediately in the admin panel.

**Default login:** `admin` / `admin` — change immediately after first login.

**Data:** `/DATA/PowerLabAppData/mstream/` — Music library (bind-mounted), playlists, and user database.
