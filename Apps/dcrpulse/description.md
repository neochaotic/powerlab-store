# Decred Pulse

DCRPulse is a monitoring dashboard for the Decred blockchain network. It tracks block explorers, vote status, treasury transactions, and node stats in real time. Intended for Decred stakeholders and node operators.

**First steps:** Open `http://<host>:8735` after the dcrd backend syncs to the chain tip (may take hours on first run).

**Data:** `/DATA/PowerLabAppData/dcrpulse/` — dcrd blockchain data, wallet state, and dashboard configuration.
