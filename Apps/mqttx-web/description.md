# MQTTX Web

MQTTX Web is a browser-based MQTT client for testing and debugging IoT message brokers. Supports MQTT 3.1/3.1.1/5.0, WebSocket transport, JSON payload formatting, and multiple simultaneous connections.

**First steps:** Open `http://<host>:80` and add a connection pointing to your MQTT broker.

**Data:** `/DATA/PowerLabAppData/mqttx-web/` — Stateless — connection profiles are stored in browser localStorage.
