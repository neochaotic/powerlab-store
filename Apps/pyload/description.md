# PyLoad

pyLoad is a download manager for one-click hosters (Mega, RapidGator, etc.), YouTube, and direct links. Features a web UI, plugin system, and unrar/decryption support. The older stable release.

**First steps:** Open `http://<host>:8000` — default login is `admin` / `pyload`. Change in the user settings.

**Default login:** `admin` / `pyload` — change immediately after first login.

**Data:** `/DATA/PowerLabAppData/pyload/` — Downloaded files and download queue database.
