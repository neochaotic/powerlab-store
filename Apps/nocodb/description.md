# NocoDB

NocoDB turns any database (MySQL, PostgreSQL, SQLite, MSSQL) into a smart spreadsheet. It provides a no-code table UI, forms, views, API generation, and role-based access — a self-hosted Airtable alternative.

**First steps:** Open `http://<host>:8080` and create the first superadmin account on the sign-up screen.

**Data:** `/DATA/PowerLabAppData/nocodb/` — NocoDB metadata database and attached file uploads.
