# n8n

n8n is a self-hostable workflow automation platform with 400+ integrations (Slack, GitHub, databases, HTTP, AI, etc.). Build complex automation flows with a visual node editor — similar to Zapier but with full code access and no per-action pricing.

**First steps:** Open `http://<host>:5678` and create your owner account on first launch.

**Data:** `/DATA/PowerLabAppData/n8n/` — SQLite database with workflows, credentials, and execution history.
