# Wallabag

Wallabag is a self-hosted read-it-later service. Save articles from any website, strip ads and clutter, and read them offline via the web UI or mobile apps. Supports Pocket/Instapaper import and OPDS export.

**First steps:** Open `http://<host>:25661` — default login is `wallabag` / `wallabag`. Change immediately in User Settings.

**Default login:** `wallabag` / `wallabag` — change immediately after first login.

**Data:** `/DATA/PowerLabAppData/wallabag/` — PostgreSQL database with saved articles, annotations, and tags.
