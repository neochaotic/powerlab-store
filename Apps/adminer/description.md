# Adminer

Adminer is a single-file database admin UI that connects to MySQL, MariaDB, PostgreSQL, SQLite, MongoDB, and more. Lighter and faster than phpMyAdmin. Designed for developers who need quick browser-based DB access.

**First steps:** Open `http://<host>:8080` and enter your database server address, credentials, and select the database.

**Data:** `/DATA/PowerLabAppData/adminer/` — Stateless — no persistent data (connects to an external DB you supply).
