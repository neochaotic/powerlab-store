# Campfire

Campfire is a minimalist, real-time group chat app designed for small teams. No accounts, no apps to install — just share a link and start chatting. Rooms are ephemeral unless persistence is enabled.

**First steps:** Open `http://<host>:80`, enter a room name to create or join it.

**Data:** `/DATA/PowerLabAppData/campfire/` — Optional SQLite persistence for chat history (ephemeral by default).
