# Lidarr

Lidarr is an automated music collection manager. It monitors release dates for your followed artists, requests downloads from your configured indexers, and organizes files using MusicBrainz metadata.

**First steps:** Open `http://<host>:8686` → Settings → Media Management to configure your music root folder, then add indexers and download clients.

**Data:** `/DATA/PowerLabAppData/lidarr/` — SQLite database with artist/album metadata, download history, and library index.
