# ActualBudget

ActualBudget is a local-first personal finance manager built around zero-based budgeting. All data stays on your server — no cloud sync, no subscriptions. Works in any modern browser with optional multi-user support.

**First steps:** Open `http://<host>:15006` after install and create your first budget.

**Data:** `/DATA/PowerLabAppData/actualbudget/` — SQLite database containing all budgets, transactions, and accounts.
