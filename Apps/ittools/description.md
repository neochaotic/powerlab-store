# IT-Tools

IT Tools is a collection of handy utilities for developers and sysadmins: JWT decoder, hash generators, base64 encoder, UUID generator, regex tester, IP calculator, and dozens more. Runs entirely in the browser — no server-side processing.

**First steps:** Open `http://<host>:80` and pick a tool from the sidebar.

**Data:** `/DATA/PowerLabAppData/ittools/` — Stateless — no data is stored.
