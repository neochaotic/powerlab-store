# Termix

Termix provides a browser-based terminal (xterm.js) connected to a server-side shell. Access a bash/zsh session on your server from any browser — useful when SSH is unavailable or inconvenient.

**First steps:** Open `http://<host>:8080` — a terminal session starts immediately.

**Data:** `/DATA/PowerLabAppData/termix/` — Stateless — no persistent shell state.
