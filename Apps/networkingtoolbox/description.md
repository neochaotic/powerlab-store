# Networking Toolbox

Networking Toolbox is a browser-based collection of network diagnostics: ping, traceroute, DNS lookup, port scanner, Whois, SSL certificate inspector, and more. Runs server-side so it tests from your server's network perspective.

**First steps:** Open `http://<host>:3000` and pick a tool.

**Data:** `/DATA/PowerLabAppData/networkingtoolbox/` — Stateless — no persistent data.
