# Uptime Kuma

Uptime Kuma is a self-hosted monitoring tool with a beautiful status dashboard. Monitors HTTP, TCP, ping, DNS, and more — with notifications via Telegram, Slack, Discord, email, webhooks, and 90+ other providers.

**First steps:** Open `http://<host>:3001` and create the admin account on the welcome screen.

**Data:** `/DATA/PowerLabAppData/uptime-kuma/` — SQLite database with monitors, status history, and notification config.
