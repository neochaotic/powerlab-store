# Grafana

Grafana is the industry-standard open-source observability platform for building dashboards from time-series data (Prometheus, InfluxDB, Loki, etc.). Used for infrastructure monitoring, application metrics, and log exploration.

**First steps:** Open `http://<host>:3003` — default login is `admin` / `admin`. You are prompted to change the password on first login.

**Default login:** `admin` / `admin` — change immediately after first login.

**Data:** `/DATA/PowerLabAppData/grafana/` — Grafana SQLite database with dashboards, data sources, and user accounts.
