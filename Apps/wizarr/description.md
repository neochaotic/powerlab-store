# Wizarr

Wizarr is an invite management system for Jellyfin, Plex, and Emby. Generate invitation links that automatically create accounts on your media server when accepted — complete with onboarding instructions and expiry controls.

**First steps:** Open `http://<host>:5690` and complete the setup wizard to connect your media server and create the first admin account.

**Data:** `/DATA/PowerLabAppData/wizarr/` — SQLite database with invitations, user records, and media server config.
