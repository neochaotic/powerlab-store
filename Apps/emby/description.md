# Emby

Emby is a media server for personal movies, TV, music, and photos. It transcodes on the fly for any device, supports multiple users with per-user libraries, and provides native apps for most platforms.

**First steps:** Open `http://<host>:8096` and complete the setup wizard to create your admin account and add media libraries.

**Data:** `/DATA/PowerLabAppData/emby/` — Emby database (metadata, users, play state) and transcoding cache.
