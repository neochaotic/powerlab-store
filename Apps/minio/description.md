# MinIO

MinIO is a high-performance, S3-compatible object storage server. Runs on commodity hardware and exposes the full AWS S3 API, making it a drop-in backend for any S3-compatible app. Includes a web console for bucket management.

**First steps:** Open `http://<host>:9010` (web console) — default credentials are `minioadmin` / `minioadmin`. Change immediately in the Identity → Users section.

**Default login:** `minioadmin` / `minioadmin` — change immediately after first login.

**Data:** `/DATA/PowerLabAppData/minio/` — Object data, bucket metadata, and IAM configuration.
