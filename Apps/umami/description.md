# Umami

Umami is a privacy-focused, GDPR-compliant web analytics platform. A self-hosted Google Analytics alternative — no cookies required, no PII collected, simple page-view and event tracking via a lightweight script.

**First steps:** Open `http://<host>:25727` — default login is `admin` / `umami`. Change immediately in Profile settings.

**Default login:** `admin` / `umami` — change immediately after first login.

**Data:** `/DATA/PowerLabAppData/umami/` — PostgreSQL database with website events and user accounts.
