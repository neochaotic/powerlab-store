# ThinkDashboard

ThinkDashboard is a personal productivity dashboard combining a todo list, notes, bookmarks, and a Pomodoro timer in a single browser tab. Designed as a browser start page replacement.

**First steps:** Open `http://<host>:8080` and start adding widgets.

**Data:** `/DATA/PowerLabAppData/thinkdashboard/` — SQLite database with todos, notes, and bookmarks.
