# BentoPDF

BentoPDF is a browser-based PDF toolkit for everyday operations: merge, split, compress, rotate, and convert. No file upload to external servers — everything runs locally in your browser via WebAssembly.

**First steps:** Open `http://<host>:8080` and drag-and-drop your PDF to start.

**Data:** `/DATA/PowerLabAppData/bentopdf/` — Stateless — no files are stored server-side.
