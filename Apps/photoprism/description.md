# PhotoPrism

PhotoPrism is an AI-powered photo management server with automatic classification, face recognition, EXIF parsing, map view, and a Google Photos-like UI. Works with your existing folder structure and supports RAW formats.

**First steps:** Open `http://<host>:2342` — default login is `admin` / `insecure`. Change the password immediately via the avatar menu.

**Default login:** `admin` / `insecure` — change immediately after first login.

**Data:** `/DATA/PowerLabAppData/photoprism/` — Photo originals, sidecar metadata, cache thumbnails, and SQLite/MariaDB index.
