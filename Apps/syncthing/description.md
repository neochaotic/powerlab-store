# Syncthing

Syncthing synchronizes files between your devices continuously, encrypted, and in real time — without a central server or cloud. Devices connect peer-to-peer via relay servers when not on the same LAN.

**First steps:** Open `http://<host>:8384` — the web UI requires no login by default (add a password in Settings → GUI). Add remote devices using their Device ID.

**Data:** `/DATA/PowerLabAppData/syncthing/` — Synced folder contents and Syncthing configuration/database.
