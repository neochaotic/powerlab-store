# Uptime Kuma

Uptime Kuma (alternate config) is a self-hosted uptime monitor with a polished status page. Same app as `uptime-kuma` — use this entry if you need a separate instance or different port binding.

**First steps:** Open `http://<host>:3001` and create the admin account on the welcome screen.

**Data:** `/DATA/PowerLabAppData/uptimekuma/` — SQLite database with monitors, status history, and notification config.
