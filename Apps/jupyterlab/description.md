# JupyterLab

JupyterLab is the next-generation web interface for Jupyter notebooks. Supports Python, R, Julia, and more. Includes a file browser, terminal, text editor, and the classic notebook interface alongside the modern panel layout.

**First steps:** Open `http://<host>:8888` — the access token is printed to the container log on first start.

**Data:** `/DATA/PowerLabAppData/jupyterlab/` — Notebooks, datasets, and installed Python packages.
