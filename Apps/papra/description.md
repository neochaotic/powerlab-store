# Papra

Papra is a document management system for digitizing and organizing paper documents. Upload scans, extract text via OCR, add tags, and find anything with full-text search. A self-hosted alternative to Paperless-ngx.

**First steps:** Open `http://<host>:1221` and create your account.

**Data:** `/DATA/PowerLabAppData/papra/` — Document files, OCR text index, and metadata database.
