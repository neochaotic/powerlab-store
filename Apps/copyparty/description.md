# copyparty

Copyparty is a self-hosted file server with a polished web UI, resumable uploads, media player, image gallery, and full-text search. Supports WebDAV and optional authentication. Runs entirely in Python with no database.

**First steps:** Open `http://<host>:3923` — anonymous access is enabled by default; add users in the config to restrict.

**Data:** `/DATA/PowerLabAppData/copyparty/` — Uploaded files and optional SQLite index for search.
