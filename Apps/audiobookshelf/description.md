# Audiobookshelf

Audiobookshelf is a self-hosted audiobook and podcast server with a polished web UI and mobile apps (iOS/Android). Supports Audible metadata matching, playback tracking, and multiple libraries across users.

**First steps:** Open `http://<host>:13378` and create the first admin account on the welcome screen.

**Data:** `/DATA/PowerLabAppData/audiobookshelf/` — Libraries, metadata cache, user progress, and podcast downloads.
