# qBittorrent

qBittorrent is a feature-rich BitTorrent client with a built-in search engine, RSS downloader, sequential downloading, IP filtering, and a clean web UI. The most popular *arr-compatible torrent client.

**First steps:** Open `http://<host>:8181` — default login is `admin` and the temporary password is printed to the container log on first start (qBittorrent 4.6+).

**Data:** `/DATA/PowerLabAppData/qbittorrent/` — Torrent files, downloaded content, and client settings.
