# pyload-ng

pyLoad NG is the actively maintained next-generation rewrite of pyLoad. Download from one-click hosters, YouTube, and direct URLs via a browser UI. Improved plugin API and Python 3 native.

**First steps:** Open `http://<host>:8000` — default login is `admin` / `pyload`. Change in the user settings.

**Default login:** `admin` / `pyload` — change immediately after first login.

**Data:** `/DATA/PowerLabAppData/pyload-ng/` — Downloaded files and download queue database.
