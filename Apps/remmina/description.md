# Remmina

Remmina is a full-featured remote desktop client (RDP, VNC, SSH, SPICE) running inside Docker, accessible via a browser-based UI. Useful for accessing Windows machines or other servers from a browser without installing a local client.

**First steps:** Open `http://<host>:3000` — the Remmina desktop loads in your browser. Add a connection and enter the target host's credentials.

**Data:** `/DATA/PowerLabAppData/remmina/` — Saved connection profiles and Remmina configuration.
