# HomeBox

Homebox is a home inventory management system for tracking items, locations, and purchases. Useful for insurance documentation, moving, or simply knowing where things are. Supports custom fields, labels, and QR codes.

**First steps:** Open `http://<host>:7745` and create the first admin account.

**Data:** `/DATA/PowerLabAppData/homebox/` — SQLite database with items, locations, labels, and attachments.
