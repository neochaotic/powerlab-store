# Resilio-sync

Resilio Sync (formerly BitTorrent Sync) uses a peer-to-peer protocol to synchronize folders between devices without a central server. Encrypted in transit, works across LAN and WAN, and supports selective sync.

**First steps:** Open `http://<host>:55555` and add your first sync folder using a secret/link from another Resilio device.

**Data:** `/DATA/PowerLabAppData/resilio-sync/` — Synchronized folder contents and peer configuration.
