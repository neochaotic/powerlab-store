# Chromium

Chromium runs a full browser instance inside Docker, accessible via a browser-based VNC/WebRTC interface (KasmVNC). Useful for isolated browsing sessions, web automation, or accessing services that require a visual browser.

**First steps:** Open `http://<host>:3000` — the browser desktop loads directly in your browser tab.

**Data:** `/DATA/PowerLabAppData/chromium/` — Browser profile, bookmarks, and cached sessions.
