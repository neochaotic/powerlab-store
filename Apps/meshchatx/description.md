# MeshChatX

MeshChatX is a decentralized group chat app designed for mesh networks and offline/local-area scenarios. Nodes discover each other via mDNS and relay messages peer-to-peer without a central server.

**First steps:** Open `http://<host>:8000`, enter a display name, and join a room — other nodes on the same network auto-discover.

**Data:** `/DATA/PowerLabAppData/meshchatx/` — In-memory message history (no persistence by default).
