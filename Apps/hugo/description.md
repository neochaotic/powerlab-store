# Hugo

Hugo is the world's fastest static site generator. This instance runs the Hugo development server, which renders your content in real time and serves it locally. Typically used for previewing before deploying to a CDN.

**First steps:** Open `http://<host>:1313` to see the live preview. Mount your Hugo site content into `/src`.

**Data:** `/DATA/PowerLabAppData/hugo/` — Your Hugo site source files (bind-mounted).
