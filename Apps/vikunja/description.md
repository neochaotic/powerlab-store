# Vikunja

Vikunja is a self-hosted task management app with projects, tasks, subtasks, labels, due dates, Gantt charts, and a Kanban board. Supports teams with per-project sharing and a REST API. A self-hosted Todoist/TickTick.

**First steps:** Open `http://<host>:3456` and register the first user (becomes admin).

**Data:** `/DATA/PowerLabAppData/vikunja/` — PostgreSQL database with tasks, projects, and user data.
