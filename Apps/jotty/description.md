# Jotty

Jotty is a minimal, distraction-free note-taking app. No folders, no tags — just a running list of plain-text notes with instant search. Designed for quick captures that don't belong in a complex knowledge base.

**First steps:** Open `http://<host>:3000` and start writing.

**Data:** `/DATA/PowerLabAppData/jotty/` — SQLite database with notes.
