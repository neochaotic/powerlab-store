# DeepSea Dashboard

DeepSea is a Decred Proof-of-Stake solo staking management tool. It helps you purchase and manage tickets, monitor voting wallets, and track treasury proposals — all from a browser UI backed by dcrwallet.

**First steps:** Open `http://<host>:8000` and connect it to your running dcrwallet instance.

**Data:** `/DATA/PowerLabAppData/deepsea/` — Staking configuration and ticket purchase history.
