# Nzbget

NZBGet is a high-performance Usenet downloader written in C++. It's faster and more CPU-efficient than SABnzbd, with a clean web UI, scripting support, and direct integration with Sonarr/Radarr.

**First steps:** Open `http://<host>:6789` — default login is `nzbget` / `tegbzn6789`. Change in Settings → Security immediately.

**Default login:** `nzbget` / `tegbzn6789` — change immediately after first login.
