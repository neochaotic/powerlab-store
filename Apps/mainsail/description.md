# Mainsail

Mainsail is the most popular web interface for Klipper-based 3D printers. It provides real-time print monitoring, temperature graphs, GCode console, webcam support, and configuration file editing from any browser.

**First steps:** Open `http://<host>:80` and configure the Moonraker API URL pointing to your Klipper instance.

**Data:** `/DATA/PowerLabAppData/mainsail/` — Mainsail configuration and cached print data (stateless against Klipper).
