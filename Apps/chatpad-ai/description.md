# Chatpad AI

ChatPad AI is a privacy-focused ChatGPT desktop-style UI that runs entirely in the browser. Conversations are stored locally in IndexedDB — nothing leaves your network. Supports OpenAI-compatible backends including local Ollama.

**First steps:** Open `http://<host>:80` and enter your API key or point to a local Ollama URL in settings.

**Data:** `/DATA/PowerLabAppData/chatpad-ai/` — Stateless server — all conversation data lives in browser localStorage.
