# ChatbotUI

Chatbot UI (alternate build, port 3080) is an open-source ChatGPT-style interface supporting multiple LLM providers including OpenAI, Anthropic, and local Ollama models. Includes folder organization and prompt library.

**First steps:** Open `http://<host>:3080` and configure your LLM provider API key in Settings.

**Data:** `/DATA/PowerLabAppData/chatbotui/` — Conversation history and prompt templates in a local database.
