# Duplicati

Duplicati is an encrypted backup client with support for cloud storage backends (S3, Backblaze, Google Drive, SFTP, and many more). Scheduled backups with incremental deduplication and AES-256 encryption.

**First steps:** Open `http://<host>:8200` and add your first backup job — choose a source, a destination, and set a passphrase.

**Data:** `/DATA/PowerLabAppData/duplicati/` — Backup job configurations, schedules, and local database index.
