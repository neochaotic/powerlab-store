# Linkwarden

Linkwarden is a collaborative bookmark manager that snapshots pages as PDFs and screenshots so links never die. Supports collections, tags, and multi-user access with per-collection sharing.

**First steps:** Open `http://<host>:3428` and register the first user (becomes admin).

**Data:** `/DATA/PowerLabAppData/linkwarden/` — PostgreSQL database, saved page snapshots, and screenshots.
