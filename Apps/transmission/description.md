# Transmission

Transmission is a lightweight, fast BitTorrent client with a clean web UI. Lower resource footprint than qBittorrent. Supports remote RPC, watchfolders, blocklists, and encryption.

**First steps:** Open `http://<host>:9091` — default login is `admin` / `admin` (set via `USER`/`PASS` env vars). Change in Settings.

**Default login:** `admin` / `admin` _(configurable via env vars)_ — change immediately after first login.

**Data:** `/DATA/PowerLabAppData/transmission/` — Torrent files, downloaded content, and settings.
