# Kiwix

Kiwix serves offline Wikipedia, Stack Overflow, Project Gutenberg, and other web content as ZIM archives — no internet connection required. Ideal for air-gapped environments, schools, or bandwidth-constrained deployments.

**First steps:** Open `http://<host>:8080` and download ZIM content files via the built-in library browser.

**Data:** `/DATA/PowerLabAppData/kiwix/` — ZIM archive files (Wikipedia, etc.) and library index.
