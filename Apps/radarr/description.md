# Radarr

Radarr is an automated movie collection manager. It monitors release calendars, grabs downloads from configured indexers when movies become available, and renames/organizes your library using custom naming formats.

**First steps:** Open `http://<host>:7878` → Settings → Media Management to set your movie root folder, then add indexers and download clients.

**Data:** `/DATA/PowerLabAppData/radarr/` — SQLite database with movie metadata, download history, and library index.
