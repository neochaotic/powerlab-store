# Ombi

Ombi is a media request and user management system for Plex, Emby, and Jellyfin. Users can request movies and TV shows; Ombi pushes approved requests to Radarr/Sonarr automatically.

**First steps:** Open `http://<host>:3579` and complete the setup wizard to connect your media server and *arr instances.

**Data:** `/DATA/PowerLabAppData/ombi/` — SQLite database with request history, user accounts, and notification settings.
