# Homarr

Homarr is a sleek, customizable home server dashboard with drag-and-drop layout, integrations (Sonarr, Radarr, qBittorrent, Plex, etc.), search widgets, and a Docker container overview. A polished alternative to Heimdall.

**First steps:** Open `http://<host>:7575` and customize your board — tiles, widgets, and integrations.

**Data:** `/DATA/PowerLabAppData/homarr/` — Dashboard configuration, icon cache, and integration credentials.
