# Mazanoke

Mazanoke is a browser-based image optimizer. It compresses JPEG, PNG, WebP, and AVIF images using efficient algorithms directly in the browser — no server upload required. Useful for optimizing assets before publishing.

**First steps:** Open `http://<host>:80` and drop your images to compress them.

**Data:** `/DATA/PowerLabAppData/mazanoke/` — Stateless — no images are stored.
