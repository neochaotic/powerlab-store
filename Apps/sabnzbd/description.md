# Sabnzbd

SABnzbd is a full-featured Usenet downloader with a polished web UI, NZB auto-unpack, repair, and direct integration with Sonarr/Radarr. Python-based and highly configurable.

**First steps:** Open `http://<host>:8282` and complete the quick-start wizard to add a Usenet server and set your download folder.

**Data:** `/DATA/PowerLabAppData/sabnzbd/` — Downloaded NZB content, incomplete downloads, and client configuration.
