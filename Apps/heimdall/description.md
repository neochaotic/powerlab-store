# Heimdall

Heimdall is a simple, elegant application dashboard for your self-hosted services. Pin your apps as large tiles with icons, optionally show live stats from supported apps (Sonarr, Radarr, etc.), and access everything from one page.

**First steps:** Open `http://<host>:80` and click the `+` button to add your first application tile.

**Data:** `/DATA/PowerLabAppData/heimdall/` — SQLite database with tile configuration and settings.
