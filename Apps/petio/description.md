# Petio

Petio is a media request and management app similar to Overseerr, but compatible with both Plex and Emby. Features a Netflix-style discovery UI, automated approval rules, and user notification support.

**First steps:** Open `http://<host>:7777` and complete the setup wizard.

**Data:** `/DATA/PowerLabAppData/petio/` — MongoDB database with requests, settings, and user data.
