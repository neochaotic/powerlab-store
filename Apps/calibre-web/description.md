# Calibre-web

Calibre-Web provides a clean browser interface for your Calibre e-book library, including online reading (EPUB), download, metadata editing, and OPDS catalog for e-readers. Requires an existing Calibre library directory.

**First steps:** Open `http://<host>:8083` — default login is `admin` / `admin123`. Change immediately in User Settings.

**Default login:** `admin` / `admin123` — change immediately after first login.

**Data:** `/DATA/PowerLabAppData/calibre-web/` — Calibre library path (bind-mounted), reading progress, and user accounts in a SQLite DB.
