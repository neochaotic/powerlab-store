# Stirling PDF

Stirling-PDF is a self-hosted, all-in-one PDF toolkit with 40+ tools: merge, split, compress, OCR, sign, watermark, convert, and more. Nothing leaves your server — all operations run locally.

**First steps:** Open `http://<host>:8080` and pick a tool from the sidebar.

**Data:** `/DATA/PowerLabAppData/stirling-pdf/` — Temporary processing workspace (output files are returned immediately, not stored).
