# Overseerr

Overseerr is a modern media request platform for Plex users. It provides a polished interface for browsing TMDB content, requesting downloads, and managing approvals — with Radarr/Sonarr integration and mobile-friendly design.

**First steps:** Open `http://<host>:5055` and complete the setup wizard to sign in with your Plex account and connect *arr instances.

**Data:** `/DATA/PowerLabAppData/overseerr/` — SQLite database with requests, user accounts, and notification settings.
