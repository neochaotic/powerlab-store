# code-server

code-server runs VS Code in the browser, giving you a full IDE accessible from any device. Supports extensions, integrated terminal, Git, and the full VS Code feature set. Ideal for remote development or coding from a tablet.

**First steps:** Open `http://<host>:8080` — the password is printed to the container log on first start (or set via `PASSWORD` env var).

**Data:** `/DATA/PowerLabAppData/code-server/` — VS Code configuration, extensions, and your workspace files.
