# Autobrr

Autobrr is a modern release automation tool that monitors IRC announce channels and RSS feeds, then pushes matching releases to your download client (qBittorrent, Deluge, SABnzbd, etc.). Replaces autodl-irssi without requiring a full IRC client.

**First steps:** Open `http://<host>:7474` and create the admin account on first launch.

**Data:** `/DATA/PowerLabAppData/autobrr/` — SQLite database with filters, indexers, and release history.
