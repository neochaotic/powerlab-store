# InfluxDB 2

InfluxDB 2 is a purpose-built time-series database for metrics, events, and IoT data. It includes a built-in web UI (Chronograf), Flux query language, and a dashboard builder. Commonly paired with Telegraf and Grafana.

**First steps:** Open `http://<host>:8086` and complete the onboarding wizard to create your org, bucket, and admin credentials.

**Data:** `/DATA/PowerLabAppData/influxdb2/` — Time-series data, dashboards, and user configuration.
