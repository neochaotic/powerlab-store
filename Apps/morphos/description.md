# Morphos server

Morphos is a self-hosted file format converter supporting documents, images, audio, and video. Upload a file, pick the output format, download the result. Powered by LibreOffice and FFmpeg under the hood.

**First steps:** Open `http://<host>:8080`, upload a file, and select the target format.

**Data:** `/DATA/PowerLabAppData/morphos/` — Temporary conversion workspace (files are not permanently stored).
