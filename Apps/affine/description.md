# Affine

AFFiNE is a next-generation knowledge base that combines docs, whiteboards, and databases in one workspace. Supports real-time collaboration, rich markdown, and edgeless canvas mode. A privacy-friendly alternative to Notion.

**First steps:** Open `http://<host>:3010` and create your workspace. First user becomes the admin.

**Data:** `/DATA/PowerLabAppData/affine/` — PostgreSQL data, user uploads, and blob storage.
