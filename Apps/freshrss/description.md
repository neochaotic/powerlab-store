# FreshRSS

FreshRSS is a self-hosted RSS aggregator with a clean UI, multi-user support, fever/Google Reader API compatibility, and mobile apps (Reeder, FeedMe, Fluent Reader). A full-featured replacement for Google Reader.

**First steps:** Open `http://<host>:80` and complete the installation wizard to create your admin account.

**Data:** `/DATA/PowerLabAppData/freshrss/` — PostgreSQL or SQLite database with feeds, articles, and user preferences.
