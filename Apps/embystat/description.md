# Embystat

EmbyStat aggregates statistics from your Emby server into detailed dashboards: library counts, watched vs. unwatched, user activity, duplicate detection, and more. Read-only — it never modifies your Emby library.

**First steps:** Open `http://<host>:6555` and enter your Emby server URL + API key to connect.

**Data:** `/DATA/PowerLabAppData/embystat/` — Local stats cache (SQLite) synced from your Emby server.
