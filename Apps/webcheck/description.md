# WebCheck

Web Check is an all-in-one OSINT tool for websites: DNS records, WHOIS, SSL certificate details, open ports, security headers, technologies detected, and more — all from a single URL input.

**First steps:** Open `http://<host>:3000`, enter any URL to get a full report.

**Data:** `/DATA/PowerLabAppData/webcheck/` — Stateless — results are fetched in real time, not stored.
