# Unifi-controller

UniFi Network Server (formerly UniFi Controller) is the management software for Ubiquiti UniFi access points, switches, and gateways. Adopt and configure your UniFi hardware from a single dashboard.

**First steps:** Open `http://<host>:8080` — you will be redirected to HTTPS on port 8443. Complete the setup wizard to create your admin account and adopt your UniFi devices.

**Data:** `/DATA/PowerLabAppData/unifi-controller/` — MongoDB database with device configuration, site topology, and historical stats.
