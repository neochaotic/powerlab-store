# OpenCode

OpenCode is a terminal-based AI coding assistant with a web UI. It integrates with LLM providers (OpenAI, Anthropic, local Ollama) to provide code generation, review, and refactoring through a browser interface.

**First steps:** Open `http://<host>:4096` and configure your LLM provider API key.

**Data:** `/DATA/PowerLabAppData/opencode/` — Session history and configuration.
