# Bazarr

Bazarr automatically downloads subtitles for movies and TV shows managed by Radarr and Sonarr. Supports 50+ subtitle providers including OpenSubtitles, Subscene, and embedded OCR. Integrates directly into the *arr media stack.

**First steps:** Open `http://<host>:6767` → Settings → Connect Sonarr/Radarr with their API keys.

**Data:** `/DATA/PowerLabAppData/bazarr/` — SQLite config, subtitle cache, and provider credentials.
