# Firefox

Firefox runs inside Docker and is served via a browser-based VNC/WebRTC interface (KasmVNC). Useful for isolated browsing, testing web apps behind your VPN, or browsing from a device without a full OS.

**First steps:** Open `http://<host>:3000` — the full Firefox UI loads in your browser tab.

**Data:** `/DATA/PowerLabAppData/firefox/` — Firefox profile including bookmarks, history, and cookies.
