# Readarr

Readarr is an automated e-book and audiobook manager, part of the *arr family. It monitors Goodreads for new releases by your followed authors and downloads them via Usenet or torrent.

**First steps:** Open `http://<host>:8787` → Settings → Media Management to set your book root folder, then configure indexers and download clients.

**Data:** `/DATA/PowerLabAppData/readarr/` — SQLite database with book/author metadata and download history.
